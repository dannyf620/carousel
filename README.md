# RCNCarousel

Esete proyceto fue generado con  [Angular CLI](https://github.com/angular/angular-cli) version 1.4.3.

## Development server

para instalar todos los complementos en necesario user el comando  `npm install`
Correr `ng serve` para lanzar un servidor . navegar a  `http://localhost:4200/`. La aplicación se volverá a cargar automáticamente si cambia alguno de los archivos fuente.
